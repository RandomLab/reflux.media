# Hundred Rabbits


Hundred Rabbits (100r) est un duo/collectif d'artistes queers vivant et travaillant ensemble sur un voilier sur l'océan pacifique. Iels portent une réflexion sur les limites d'un monde hyper-connecté et gourmant en ressource ainsi que sur les réponses low-tech à leurs besoins à bord. À ce jour iels ont développé 8 outils numériques et 7 jeux vidéos dont la plutpart sont gratuits.


## UXN
Toujours dans une optique d'être moins gourmants en ressources, iels ont créé l’écosystème UXN, un "pile de calcul à l'état pur" qui s’apparente à un système d’exploitation pour petites machines (vieilles consoles portables, petits ordinateurs, etc..) qu'on peut aussi utiliser sous MAC, Linux ou Windows via un émulateur.

## ORCA

Orca est un séquenceur midi experimental qui permet de créer un signal midi à partir d'ASCII. La prise en main est assez compliquée puisque l’interface et le code ne ressemble à aucun autre outil mais la documentation github et les ressources sur youtube sont assez nombreuses pour un apprentissage efficace. Orca est compatible avec SonicPi, logiciel très utilisé dans le live coding et peut aussi servir à la génération de graphismes connectés au son.
Petit tips pour commencer : installez directement Orca, Pilot et Cassetter, les trois fonctionnent en écosystème.

## DotGrid et Ronin
 100r ont aussi développés plusieurs outils de graphisme, Ronin et DotGrid en sont deux exemples qui s'utilisent directement dans le navigateur. Ronin est un outil de code similaire à Processing qui peut amener à du live coding graphique et DotGrid est un outil vectoriel ultra simplifié avec lequel ont été dessiné tous les logos de 100r.
