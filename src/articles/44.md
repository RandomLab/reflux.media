# *Minecraft*, dans *Minecraft*

Le joueur Sammyuri a crée un ordinateur dans le jeu vidéo *Minecraft*. Pour cela il utilise un logiciel sur openredstone.org premettant de recréer un PC entieremment fonctionnel au sein même de jeu. Si pour l'instant l'ordinateur ne permet de faire tourner qu'une version allégée de *Minecraft*, il projette maintenant de pouvoir jouer à DOUM dans *Minecraft*...
