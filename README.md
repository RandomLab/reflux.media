# reflux.media

# installation
- cloner le dépôt git avec `git clone git@github.com:RandomLab/reflux.media.git`
- se positioner dans le dossier avec `cd reflux.media`
- installer les dépendances avec `npm install`

# lancement
- npm run build : enregistre les articles depuis wiki.reflux.media en markdown
- npm run generate : génère les fichiers statiques html
